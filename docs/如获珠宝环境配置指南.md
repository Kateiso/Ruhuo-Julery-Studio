# 如获珠宝·智能视频工坊 - 环境配置指南

## 快速开始

### macOS 用户

**方法 1：双击 .app 启动（推荐）**
1. 打开 `app/如获珠宝视频工坊.app`
2. 首次运行会提示安装依赖（如未安装）

**方法 2：命令行启动**
```bash
cd MoneyPrinterPlus
bash start.sh
```

---

## 环境配置

### 系统要求
- macOS 10.15+ / Windows 10+ / Linux
- Python 3.11（推荐）
- 8GB+ RAM

### 安装步骤

1. **创建虚拟环境**
```bash
python3.11 -m venv venv
source venv/bin/activate  # macOS/Linux
# venv\Scripts\activate   # Windows
```

2. **安装依赖**
```bash
pip install -r requirements.txt
```

3. **配置 API 密钥**
- 复制 `config/config.example.yml` 为 `config/config.yml`
- 填写通义千问 API Key、云端语音服务密钥等

4. **启动应用**
```bash
bash start.sh        # macOS/Linux
# start.bat          # Windows
```

## 项目结构

```
MoneyPrinterPlus/
├── app/                    # .app 应用包
├── pages/                  # Streamlit 页面
│   ├── 00_script_generator.py  # 拍摄脚本生成器（多轮对话）
│   ├── 01_auto_video.py        # AI 视频生成
│   └── common.py               # 公共组件
├── services/               # 业务逻辑服务
│   ├── llm/                # LLM 服务（通义千问等）
│   └── audio/              # 语音服务
├── styles/custom.css       # 爱马仕风格样式
├── config/                 # 配置文件
├── gui.py                  # Streamlit 入口
└── start.sh                # 启动脚本
```
